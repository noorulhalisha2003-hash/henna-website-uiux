##  Project Overview
A business website UI for a henna artist to showcase designs and allow customers to book appointments.

##  Objective
- Improve online presence
- Enable easy booking
- Maintain elegant and aesthetic UI

##  User Flow
Home → Gallery → Booking → Confirmation

##  Key Features
- Booking form
- Gallery layout
- Clean typography
